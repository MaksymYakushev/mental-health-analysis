# Mental Health Dashboard

## Project overview

This Tableau dashboard provides an interactive analysis of mental health patterns in a remote work environment, offering insights into stress levels, burnout, productivity, and work-life balance across different demographics and regions.

## Business Context

Companies and organizations increasingly monitor employee wellbeing to understand the impact of remote work on mental health and productivity. This project visualizes these patterns, allowing stakeholders to explore:

- How stress, burnout, and work-life balance vary across employees
- Whether remote, hybrid, or onsite work affects mental wellbeing
- How factors like workload, sleep, and physical activity relate to mental health
- Differences across ages and demographic groups
